# TetrisStorm
A Friendly tetris game built as a team

# Startup
1) Install yarn from https://yarnpkg.com
2) Install node from https://nodejs.org/en/
3) Restart pc
4) Run Yarn install on client directory
5) Run Yarn start to startup the app
